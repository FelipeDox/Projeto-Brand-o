# Projeto-Brand-o
 Projeto de um site voltado para atender a demanda de instalação de placas voltaicas e produção de energia solar. Neste site, reuniremos informações e funções necessárias para que um cliente marque uma visita em sua residência/empresa para a instalação de placa voltaica, o dimensionamento da instalação e a necessidade energética.

 Na primeira semana do projeto, definimos a aparência geral do site, incluindo o estilo visual e a paleta de cores. Após essa etapa, iniciamos o desenvolvimento do código. Considerando que o tema do site é energia solar, optamos por utilizar cores claras, como amarelo, verde claro e branco para o fundo, reforçando a identidade visual proposta.

No front-end, trabalhamos na criação de abas de navegação que facilitam o acesso do usuário às principais áreas do site: página inicial, contato, fornecedores e tipos de serviço. Essas seções foram estruturadas para garantir uma experiência de navegação simples, intuitiva e eficiente.
